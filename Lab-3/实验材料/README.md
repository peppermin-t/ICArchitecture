# 文件说明

## vivado

- proj文件夹存放vivado工程
- rtl文件夹存放设计文件
- tb文件夹存放仿真激励文件

- test.py用于测试矩阵乘结果是否正确

## linux_transplantation

文件过大，见[下载链接](https://bhpan.buaa.edu.cn:443/link/92D839CD3C53F031AE06FE63ADE58B36)

- boot.zip为linux系统移植所需的启动和镜像文件
- linaro-jessie-alip-20161117-32.tar为linux根文件系统
- putty.exe为串口通信工具
